🖥️ Windows Intranet Suit

A LAN-First Offline Cloud Built by a Man and a Bot

📖 Origin Story

This project began with a spark — a YouTube video showing the Kiwix app and its offline capabilities. That moment revealed something powerful:

“LAN can be more than just a network — it can be your own internet.”

At the heart of this project was curiosity, fueled by a childhood memory — a 12th-grade computer teacher introducing Python. Years later, that small seed grew into something remarkable.

🤝 Built by Two Minds

  • 🧠 A Man — who had the vision, ideas, and persistence
  • ⚙️ A Bot — who turned those ideas into code and structure

Together, they built a modular, efficient, self-hosted system — entirely offline, with full UI control, smart fallback pages, themed interfaces, and domain-based routing.

🧰 What It Includes

  • Offline Encyclopedia (Kiwix)
  • Media Streaming (Jellyfin)
  • Music Library (Navidrome)
  • Comics Reader (Komga)
  • Offline Translator (LibreTranslate)
  • Games (HTML Games server)
  • Local Chatbot (Qwen via Ollama)
  • Custom Dashboards (.exe with GUI)
  • LAN Routing (NGINX + Technitium)
  • Fallback Page (offline.html)
  • An HomePage (LanScout)

Each service has its own launcher, logs, icon, and LAN domain.

🌐 Why It Matters

This system runs on an old i3 processor with 12GB RAM. No cloud. No external dependencies.

It's proof that you don’t need modern hardware to build a modern network. All you need is an idea — and a tool that listens.

🧩 The Name: Windows Intranet Suit

It runs on Windows. It turns your PC into a fully offline, modular, personal cloud. And like a suit, it’s tailored to fit exactly what you need — nothing more, nothing less.
